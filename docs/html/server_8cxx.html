Detailed Description

This file contains the implementation of the server component using CivetServer.

The server component initializes and runs a local server based on the configuration specified in the server.ini file. This enables local file access through a web browser.

Warning
Ensure that the configuration file server.ini is properly formatted and located in the config directory.
