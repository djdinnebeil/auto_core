Detailed Description

Main entry point for the Auto Core application.

This file contains the primary logic for initializing and running the Auto Core application. It includes functions for setting up the console, initializing core components, creating named pipe servers for inter-process communication, and starting necessary components. The main function manages the application lifecycle, including setting up the environment, entering the message loop, and handling exceptions.
