WPARAM wParam,

export

Callback function to send numpad events.

This function processes keyboard events and sends specific messages based on the numpad key pressed.

Parameters
nCodeThe hook code. If less than zero, the hook procedure must pass the message to CallNextHookEx without further processing.
wParamThe identifier of the keyboard message.
lParamA pointer to a KBDLLHOOKSTRUCT structure.
Returns
If nCode is less than zero, the hook procedure must return the value returned by CallNextHookEx. Otherwise, it must return zero.
